# AI Voice Language Tutor

AI Voice Language Tutor is a voice-first language learning platform built to help people learn by speaking with an AI tutor, not just by memorizing vocabulary. The product is designed as a real business, with a clear learning path, paid course unlocks, exams, and completion certificates.

## Product Vision

The core promise of the platform is simple:

**Start from zero. Speak with AI. Get certified.**

The experience is built around daily speaking practice, pronunciation correction, listening improvement, and confidence building. Users should feel like they are practicing with a private speaking coach that is always available, affordable, and non-judgmental.

This product is meant to stand apart from traditional language apps by focusing on:

- voice-first learning
- pronunciation feedback
- AI conversation practice
- structured course progression
- exams at the end of each course
- verifiable completion certificates

## Supported Languages In Version 1

The first release will support:

- Japanese
- German
- Spanish
- English
- French

## Target Users

The platform is intended for:

- complete beginners starting from zero
- students preparing for basic language proficiency
- people planning to study or work abroad
- job seekers improving spoken English or another language
- self-learners who want affordable speaking practice
- shy users who want to practice with AI before speaking with real people

## Core User Journey

The planned user flow is:

1. User lands on the website and understands the promise of AI-based speaking practice.
2. User signs up with Google.
3. User selects their native language.
4. User selects the language they want to learn.
5. User starts the free `Basic 1` course.
6. User completes lessons and voice practice.
7. User takes the final exam for the course.
8. User receives a completion certificate if they pass.
9. User unlocks the next course level for `$20`.

## Product Structure

Each language follows the same progression:

- `Basic 1`
- `Basic 2`
- `Intermediate 1`
- `Intermediate 2`
- `Advanced 1`
- `Advanced 2`

### Level Strategy

- `Basic 1`: from-zero foundation course, free
- `Basic 2`: beginner communication, paid
- `Intermediate 1`: daily conversation, paid
- `Intermediate 2`: practical and workplace conversation, paid
- `Advanced 1`: professional and advanced communication, paid
- `Advanced 2`: mastery-level fluency, paid

For Japanese, levels can map to JLPT-style progression. For German, Spanish, French, and English, levels can map to CEFR-style progression.

## Pricing Model

- `Basic 1` is free for every language.
- Every level after `Basic 1` costs `$20`.

Examples:

- Japanese `Basic 1 / Pre-N5 Foundation` -> Free
- Japanese `Basic 2 / N5 Beginner Communication` -> $20
- German `Basic 1 / A1 Foundation` -> Free
- German `Basic 2 / A2 Everyday German` -> $20
- English `Basic 1 / A1 Foundation` -> Free
- English `Basic 2 / A2 Everyday English` -> $20

The product should be easy to understand commercially: one free entry course, then simple paid unlocks by level.

## Learning Philosophy

The product is built around one major rule:

**Users learn a language by talking, not only by memorizing.**

That means the system should emphasize:

- hearing the language
- repeating words and phrases
- being corrected in real time
- improving pronunciation and confidence
- building speaking ability step by step

## Basic 1 Plan

`Basic 1` is free and designed for complete beginners. It should stay simple and highly approachable.

### Basic 1 Goal

Help users build strong foundations in:

- alphabet or script
- sounds
- numbers
- greetings
- basic words
- pronunciation
- listening recognition

### Basic 1 Modules

- `Module 1`: alphabet or script
- `Module 2`: numbers
- `Module 3`: greetings
- `Module 4`: basic words
- `Module 5`: listening recognition
- `Module 6`: speak and repeat
- `Module 7`: word memory practice

For Japanese, `Basic 1` should focus on Hiragana, Katakana, and basic sounds, without heavy Kanji.

### Basic 1 Exam

The final exam should test:

- alphabet or script recognition
- numbers 1 to 100
- greetings
- vocabulary
- listening recognition
- pronunciation repeat ability

Passing score: `70%`

Example certificate:

- `Japanese Basic 1 Foundation Certificate`

## Basic 2 Plan

`Basic 2` is the first paid level and introduces sentence-level learning. This is where practical communication begins.

Users should learn simple sentences such as:

- My name is Rahul.
- I am from India.
- I like coffee.
- I want water.
- Where is the station?
- How much is this?

### Basic 2 Activities

- simple sentence building
- AI voice conversation
- pronunciation correction
- short dialogue listening
- roleplay practice
- mini speaking tests
- grammar correction

## AI Tutor Experience

The AI tutor should feel like a real speaking coach.

The tutor should:

- speak to the user
- ask questions
- listen to the user
- convert voice to text
- correct pronunciation
- correct grammar
- explain mistakes simply
- request repetition when needed
- score attempts
- track progress

Example style of feedback:

> Good attempt. Your meaning was correct, but your pronunciation was slightly unclear. Try saying it slower.

The product should make users feel safe practicing without embarrassment.

## Exam System

Every course should end with an exam. Exams should include:

- vocabulary testing
- listening testing
- pronunciation testing
- speaking testing
- grammar or sentence testing
- AI interview section

Rules:

- passing score is at least `70%`
- users can retake after practice
- a delayed free retake model can be introduced
- paid users may receive more flexible retakes

## Certificate System

Certificates are completion certificates, not official institutional or government language certifications.

Each certificate should include:

- user name
- language
- course level
- completion date
- score
- certificate ID
- QR code
- verification URL

Important business rule:

- do not present these as JLPT, Goethe, DELF, DELE, IELTS, or any official exam equivalent

## Product Rules

These rules should guide planning and development:

- `Basic 1` must stay extremely beginner-friendly
- sentence learning starts in `Basic 2`
- AI should not introduce advanced material in beginner lessons
- every lesson needs a clear objective
- every course ends with an exam
- every certificate must be verifiable
- certificates must never be marketed as official government or institutional certification
